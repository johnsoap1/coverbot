#### Mandatory Vars

- `API_ID` - Get this from [my.telegram.org](https://my.telegram.org/auth)
- `API_HASH` - Get this from [my.telegram.org](https://my.telegram.org/auth)
- `BOT_TOKEN` - Get this from [@BotFather](https://t.me/BotFather)

I added in a rate limiter to prevent the bot from being banned.
And added in a storage group to store the media sent by the user.
Album support is added in.
And added in a delete original messages feature to delete the original messages sent by the user.
